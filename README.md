# Image Processing using Convolution

A Digital Signal Processing project using Python.

## Features

* Image Blurring
* Sharpening
* Edge Detection
* GUI for Upload and Processing

## Technologies Used

* Python
* OpenCV
* NumPy
* Tkinter

## Files

* `main.py` → Direct image processing
* `main2.py` → GUI version

## How to Run

```bash
pip install opencv-python numpy matplotlib pillow
python main2.py
```
